# alanportfolio

Personal portfolio website built with **React 19**, **TypeScript**, and **Vite 6**.

> **Live at:** [www.alan19.com.np](https://www.alan19.com.np)

## Tech Stack

- **Framework:** React 19 + TypeScript
- **Build tool:** Vite 6
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion v12
- **Icons:** Lucide React
- **Dark mode:** next-themes
- **Deployment:** GitHub Pages

## Sections

- **Hero** — Profile, tagline, CTA buttons, social links
- **Stats** — Animated counters (experience, projects, technologies)
- **About** — Bio, fact cards, typewriter effect, tech stack skill bars
- **Projects** — 7 project cards with tech tags and GitHub links
- **Experience** — Learning journey timeline
- **Contact** — Contact info & form (opens Gmail compose)
- **Footer** — Navigation, projects, social, copyright

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Deploy to GitHub Pages:

```bash
npm run deploy
```
